# cctv-based-traffic-analysis-system

### 프로젝트 주제
CCTV 영상을 기반으로 한 교통사고 분석 및 불법 주정차 단속 시스템

### 프로젝트 개요
1. 프로젝트 주요 서비스 내용
   - cctv 영상을 통한 교통사고 현장 및 불법 주정차 인식 시스템 개발
   - 사고 차량 및 불법 주정차 차량 번호판 인식 개발 (OCR토대)
   - 인식된 데이터를 토대로 사고 보고서를 작성해주는 서비스 개발
   - 서비스 구현 웹 구축
